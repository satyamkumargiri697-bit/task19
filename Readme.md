# Service Booking Web App

Hello! This is my project for the Service Booking Web App assignment. I have created this website using basic HTML, CSS, and JavaScript DOM manipulation.

## What this website does:
1. **Navigation Bar:** It has a logo, simple links (Home, Services, About Us, Contact Us), a username, and a red Logout button.
2. **Browse Services:** Users can see different services like Dry Cleaning and House Cleaning with their prices.
3. **Add to Cart:** When you click the "Add Item" button, the item goes to the left side in the "Added Items" list, and the total price updates automatically.
4. **Skip Item:** If you click the "Skip Item" button, that service card becomes semi-transparent and stops working.
5. **Booking Form:** There is a form with Name, Email, and Password. The "Book Now" button becomes active only after you add an item to the cart. When you submit, it shows a "Successful" message and resets the cart.

## Files in this project:
* `index.html` - Contains the basic design and structure of the website.
* `style.css` - Contains the simple styling, colors, and layout.
* `script.js` - Contains the JavaScript functions to add items, calculate the total price, and handle form submission.
* `Readme.md` - This file with project details.

## How to run the project:
1. Download or extract the project folder.
2. Double-click or open the `index.html` file in any modern web browser (Google Chrome, Microsoft Edge, Mozilla Firefox, etc.).
3. You can now test the buttons and the booking form.
